# Images Folder

Place your custom photos here. Supported formats: JPG, JPEG, PNG, WEBP

## Folder Structure

```
images/
├── hero/
│   ├── slide-1.jpg        ← Hero background slide 1 (landscape, min 1600px wide)
│   ├── slide-2.jpg        ← Hero background slide 2 (bridal theme)
│   ├── slide-3.jpg        ← Hero background slide 3 (custom tailoring theme)
│   └── portrait.jpg       ← Right-side portrait on hero (portrait orientation)
│
├── about/
│   ├── main.jpg           ← Main large image in About section (portrait, min 800px)
│   └── accent.jpg         ← Small overlapping accent image (square, min 400px)
│
├── services/
│   ├── blouse/
│   │   └── cover.jpg      ← Blouse stitching card image
│   ├── bridal-blouse/
│   │   └── cover.jpg      ← Bridal blouse card image
│   ├── suit/
│   │   └── cover.jpg      ← Suit stitching card image
│   ├── lehenga/
│   │   └── cover.jpg      ← Lehenga stitching card image
│   ├── salwar/
│   │   └── cover.jpg      ← Salwar stitching card image
│   └── kurta/
│       └── cover.jpg      ← Kurta stitching card image
│
└── gallery/
    ├── 1.jpg              ← Gallery image 1 (Bridal Blouse)
    ├── 2.jpg              ← Gallery image 2 (Lehenga)
    ├── 3.jpg              ← Gallery image 3 (Designer Suit)
    ├── 4.jpg              ← Gallery image 4 (Blouse)
    ├── 5.jpg              ← Gallery image 5 (Anarkali)
    ├── 6.jpg              ← Gallery image 6 (Wedding Collection)
    ├── 7.jpg              ← Gallery image 7 (Embroidery Work)
    ├── 8.jpg              ← Gallery image 8 (Party Wear)
    └── 9.jpg              ← Gallery image 9 (Designer Collection)
```

## Tips
- Hero backgrounds look best in **landscape** (16:9) orientation
- Service card images look best in **4:3** or **16:9** landscape
- Gallery images can be any orientation — they auto-crop to fit
- Minimum recommended size: **800px** on the shortest side for sharpness
